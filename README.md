# WEBDATA__SCRAPING
This project is a web scraping tool built using Python, requests, and BeautifulSoup to extract product names and prices from websites. It processes and stores the data in CSV format and includes basic price monitoring logic to detect deals, demonstrating real-world applications like price tracking and automated data collection.
